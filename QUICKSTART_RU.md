# 🚀 Быстрый старт — Nail Booking Telegram Mini App

Полное руководство для запуска приложения за 5 минут.

## 📱 Что это?

Приложение для онлайн-записи к мастеру маникюра через Telegram. Клиенты могут:
- ✅ Записаться на услугу
- ✅ Выбрать удобное время
- ✅ Получить напоминания
- ✅ Отменить запись

Мастер может:
- ✅ Управлять записями
- ✅ Получать уведомления
- ✅ Видеть расписание
- ✅ Получать статистику

## 🎯 За 5 минут

### 1️⃣ Локальный запуск (для тестирования)

```bash
# Перейдите в папку проекта
cd /home/ubuntu/nail_booking_bot

# Установите зависимости
pnpm install

# Запустите приложение
pnpm dev
```

Откройте http://localhost:3000 в браузере.

### 2️⃣ Развёртывание на Railway (для продакшена)

1. **Создайте репозиторий на GitHub**
```bash
git init
git add .
git commit -m "Nail Booking App"
git push -u origin main
```

2. **Откройте https://railway.app**
   - Нажмите "New Project"
   - Выберите "Deploy from GitHub"
   - Выберите ваш репозиторий

3. **Добавьте MySQL базу**
   - Нажмите "+ Add"
   - Выберите "MySQL"

4. **Добавьте переменные окружения**
   - `TELEGRAM_BOT_TOKEN` = ваш токен
   - `ADMIN_TELEGRAM_ID` = ваш ID в Telegram

5. **Установите webhook**
```bash
curl -X POST https://api.telegram.org/bot<TOKEN>/setWebhook \
  -H 'Content-Type: application/json' \
  -d '{"url":"https://your-app.railway.app/api/telegram/webhook"}'
```

**Готово!** Приложение развёрнуто на Railway ✨

## 🔑 Первые шаги после развёртывания

### 1. Установите расписание
1. Откройте приложение
2. Авторизуйтесь как администратор
3. Перейдите в **Админ-панель** → **Расписание**
4. Установите рабочие дни и часы

### 2. Добавьте услуги
1. В админ-панели перейдите в **Услуги**
2. Нажмите **Добавить услугу**
3. Заполните данные (название, время, цена)

### 3. Настройте Telegram бота
1. Откройте BotFather в Telegram (@BotFather)
2. Выберите вашего бота
3. Нажмите "Edit Bot" → "Edit Inline Button"
4. Установите текст и URL приложения

### 4. Начните принимать записи
1. Поделитесь ссылкой на бота с клиентами
2. Клиенты нажимают кнопку в боте
3. Откроется приложение для записи
4. Вы получаете уведомление о новой записи

## 🐛 Решение проблем

### Приложение не загружается
```bash
# Проверьте, что сервер запущен
pnpm dev

# Проверьте логи
tail -f .manus-logs/devserver.log
```

### Ошибка подключения к БД
- Проверьте `DATABASE_URL`
- Убедитесь, что БД запущена
- Проверьте права доступа

### Бот не отправляет сообщения
- Проверьте `TELEGRAM_BOT_TOKEN`
- Убедитесь, что webhook установлен
- Проверьте логи приложения

### Напоминания не приходят
- Проверьте `ADMIN_TELEGRAM_ID`
- Убедитесь, что бот может отправлять сообщения
- Проверьте расписание (может быть выходной)
